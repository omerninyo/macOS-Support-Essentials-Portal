# פרק 1: התקנה, הכרה ויישור קו (Setup & Fundamentals) - Hands-on Lab (Asset D)

## דרישות קדם למעבדה (Lab Prerequisites)
על המדריך לוודא לפני תחילת השיעור כי:
1. לכל חניך יש מחשב מבוסס Apple Silicon חדש או שאופס מראש באמצעות כלי ה-Erase All Content and Settings (EACS).
2. המחשב כרגע ממתין במסך ה"שלום" של ה-Setup Assistant.
3. המחשב **אינו** משויך לשרת MDM דרך ה-Apple Business Manager, כדי למנוע את הופעת מסך ה-Remote Management במעבדה זו.
4. קיימת רשת Wi-Fi פעילה בכיתה עם גישה לשרתי אפל (לצורך מעבר מסך ה-Activation Lock/שיוך).

## תרגיל 1: השלמת Setup Assistant יזום
בתרגיל זה נבצע סימולציה של חוויית המשתמש הראשונית שניגש למחשב (Out of Box Experience).
1. הדליקו את המחשב (במידה והוא כבוי) או המשיכו ממסך ה"שלום".
2. ב"מסך השפות", בחרו ב-**English** ולאחר מכן בחרו את מיקומכם כ-**Israel**.
3. התחברו לרשת ה-Wi-Fi של הכיתה.
4. במסך ה-Data & Privacy, לחצו על **Continue**.
5. במסך ה-Migration Assistant (העברת מידע ממק ישן), בחרו **Not Now** בתחתית המסך.
6. במסך ה-Sign in with your Apple ID, בחרו **Set Up Later** (ולאחר מכן Skip בחלון האזהרה). אנו לא רוצים לקשר חשבון צרכני אישי כרגע.
7. **יצירת החשבון המקומי:** 
   * **Full Name:** Student Admin
   * **Account Name:** student
   * **Password:** password (או סיסמה פשוטה אחרת שתקבעו)
8. המשיכו ללחוץ על Continue ודלגו על הגדרות ה-Screen Time, Siri, ו-Touch ID (בחרו Set Up Later), עד שתגיעו לשולחן העבודה וה-Finder.

## תרגיל 2: חילוץ מזהי חומרה ב-GUI (הממשק הגרפי)
כאנשי תמיכה, עליכם להכיר את המכונה שעליה אתם עובדים ואת גרסת מערכת ההפעלה המדויקת שלה, באמצעות כלים מובנים במערכת.
1. לחצו על מקש ה-`Option` (Alt) במקלדת והשאירו אותו לחוץ.
2. במקביל, באמצעות העכבר (או ה-Trackpad), לחצו על סמל ה**תפוח** בפינה השמאלית העליונה של המסך.
3. בחרו באפשרות הראשונה: **System Information...** (כאשר לא לוחצים על Option, יהיה כתוב About This Mac).
4. בחלון שנפתח, תחת קטגוריית **Hardware** הראשי, אתרו ורשמו לעצמכם את שני הנתונים הבאים:
   * **Chip** (או Processor Name).
   * **Serial Number (system)**.
5. כעת, בתפריט הצדדי, גללו מטה תחת קטגוריית Network ולחצו על **Wi-Fi**.
6. חפשו בחלון הימני את ה-**MAC Address** (מספר בן 12 תווים מופרד בנקודתיים) ורשמו גם אותו.
7. צאו מה-System Information, ופתחו את הגדרות המערכת: תפוח -> **System Settings**.
8. נווטו אל **General** ובחרו ב-**About**.
9. לחצו על מספר הגרסה של macOS שמופיע (למשל 14.2) ושימו לב כיצד מוצג כעת קוד ה-Build Version בסוגריים (למשל, 23C64). זהו זיהוי הגרסה המדויק הדרוש לתמיכה.

## תרגיל 3: ניווט מערכת ויישור קו (Equalizer)
למדו כיצד להתמצא בסידור הקבצים וההגדרות של ה-Mac ולטפל בתקיעות ממשק.
1. פתחו חלון **Finder** חדש (קליק על סמל הפרצוף ב-Dock או לחיצה על `Cmd+N` כשאתם בדסקטופ).
2. בשורת התפריטים העליונה, נווטו ל-**View** ובחרו ב-**Show Path Bar**. שימו לב לשורת הנתיב שהתווספה בתחתית החלון.
3. פתחו את חלון ההגדרות: לחצו על סמל התפוח -> **System Settings**.
4. בתפריט הצדדי בחרו ב-**Trackpad** ועברו על לשוניות ה-Scroll & Zoom ו-More Gestures כדי להכיר את מחוות האצבעות שיוצגו בסרטונים.
5. כעת, נדמה מצב שבו ממשק המשתמש נתקע ונאתחל את ה-Finder באופן גרפי: החזיקו את מקש ה-`Option` (Alt) במקלדת, ולחצו לחיצה ימנית (או שתי אצבעות ב-Trackpad) על סמל ה-**Finder** ב-Dock.
6. בתפריט שנפתח, ביחרו באפשרות **Relaunch**.
7. שימו לב כיצד שולחן העבודה "מהבהב" והאייקונים נעלמים לשבריר שנייה וחוזרים - ביצעתם איפוס מוצלח לממשק!

## תרגיל אקסטרה / קצה קרחון טכני
במידה ותרצו לבצע את הפעולות שראינו (חילוץ מידע מערכת ואיפוס ה-Finder) בצורה מהירה באמצעות שורת הפקודה, תוכלו להשתמש ב-Terminal. פקודות אלו שימושיות במיוחד כשמתחברים למחשב מרחוק ב-SSH.

1. פתחו את אפליקציית ה-`Terminal` מתוך תיקיית ה-Utilities.
2. כדי להציג את המספר הסידורי ופרטי החומרה של המחשב, הקלידו את הפקודה:
   ```bash
   system_profiler SPHardwareDataType
   ```
3. כדי להציג את גרסת ה-Build המדויקת של המערכת, הריצו:
   ```bash
   sw_vers
   ```
4. במקום לבצע "Relaunch" ל-Finder דרך הממשק הגרפי, ניתן לסגור את התהליך ולאלץ אותו לעלות מחדש באמצעות הפקודה:
   ```bash
   killall Finder
   ```
